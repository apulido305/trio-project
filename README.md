# URBAN MIAMI
Urban Miami is a 3 person project created at Wyncode for "friends and family". This is an example of a ecommerce shop. In this project our goal was to create a rails app that would function as a platform for creating an online store. 

## Getting Started
These instructions will let you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites
What things you need to install the software and how to install them

Google Chrome browser
Rails 

### Installing
To View the completed project navigate to [Urban Miami](https://hidden-citadel-48758.herokuapp.com/)

###### For Development
Clone the repo to your local machine
you will need a cloudinary account [Cloudinary](https://cloudinary.com/)
Then setup your cloudinary YML file [Cloudinary](https://cloudinary.com/documentation/rails_integration#configuration)
You will need a stripe account.

```
bundle install
```
```
rake db:migrate
```
